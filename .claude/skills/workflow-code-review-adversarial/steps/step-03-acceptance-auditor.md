# Step 3: Acceptance Auditor

## 목적
Story AC 각 항목의 충족 여부를 대조하고 최종 리뷰 리포트를 작성한다.

## 작업
1. Story AC 목록을 읽는다
2. 각 AC에 대해 구현·테스트가 정확히 대응하는지 확인한다
3. 테스트 커버리지 적절성을 평가한다
4. `template.md` 형식으로 최종 리뷰 리포트를 작성한다
5. PR comment로 게시한다

## 완료 조건
- [ ] AC 전 항목 대조 완료
- [ ] 리뷰 리포트 PR comment 게시
- [ ] HITL L2 사람 승인 대기
