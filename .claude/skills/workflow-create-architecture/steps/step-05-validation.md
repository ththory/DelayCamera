# Step 5: 아키텍처 검증

## 목적
`checklist.md`와 `/create-architecture-checklist` Skill로 설계 완성도를 검증한다.

## 작업
1. `checklist.md`를 로드하여 전 항목을 확인한다
2. PRD의 모든 FR·NFR이 설계에 반영됐는지 대조한다
3. 보안·성능·확장성 관점에서 설계를 재검토한다
4. 사용자에게 최종 설계를 보고하고 승인을 요청한다 (HITL L2)

## 완료 조건
- [ ] checklist.md 전 항목 통과
- [ ] FR·NFR 전부 반영 확인
- [ ] 사람 승인 완료 (HITL L2)
