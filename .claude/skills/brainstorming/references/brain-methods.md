# 브레인스토밍 기법 목록

## 발산 기법

| 기법 | 설명 | 적합한 상황 |
|---|---|---|
| HMW (How Might We) | "어떻게 ~할 수 있을까" 질문 생성 | 문제 재정의 |
| SCAMPER | 대체·결합·적용·수정·다른용도·제거·역전 | 기존 솔루션 개선 |
| 6 Thinking Hats | 6가지 관점 순차 탐색 | 다각도 분석 |
| Crazy 8s | 8분 안에 8개 아이디어 스케치 | 빠른 발산 |
| Worst Possible Idea | 최악의 아이디어 → 반전 | 고정관념 탈피 |
| Analogies | 다른 도메인 유사 문제 해결법 차용 | 혁신적 접근 |

## 수렴 기법

| 기법 | 설명 |
|---|---|
| 점 투표 (Dot Voting) | 아이디어별 점수 부여 |
| 영향·노력 매트릭스 | 가로: 노력, 세로: 영향 → 우선순위 |
| RICE Score | Reach·Impact·Confidence·Effort |
