# Step 2: PRD 작성

## 목적
분석 결과를 기반으로 `template.md` 형식에 맞춰 PRD를 작성한다.

## 작업
1. `template.md`를 로드하여 포맷을 확인한다
2. AI Consumer Metadata (ADR-016) frontmatter를 포함한다
3. 각 섹션을 작성한다:
   - 배경·목적
   - 사용자 스토리 목록
   - 기능 요구사항 (FR)
   - 비기능 요구사항 (NFR)
   - 제약·가정
4. `docs/prd/<ticket>.md`에 저장한다

## 완료 조건
- [ ] template.md 전 섹션 작성
- [ ] AI Consumer Metadata 포함
