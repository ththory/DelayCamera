# Edge Case 패턴 목록

## 입력 경계값
- 빈 문자열 `""`
- 공백만 있는 문자열 `"   "`
- 최솟값·최댓값 (int 오버플로우)
- 음수 입력
- 소수점 (부동소수점 정밀도)

## 컬렉션
- 빈 리스트·딕셔너리 `[]`, `{}`
- 단일 요소 컬렉션
- 중복 요소
- 매우 큰 컬렉션 (메모리·성능)

## null / None
- None 체크 없이 메서드 호출
- Optional 언래핑 실패
- undefined (JS/TS)

## 네트워크·IO
- 타임아웃 (설정 여부)
- 연결 실패·재시도 로직
- 부분 응답·스트리밍

## 동시성
- 공유 상태 변경 (레이스 컨디션)
- 데드락 가능성
- 트랜잭션 격리 수준
