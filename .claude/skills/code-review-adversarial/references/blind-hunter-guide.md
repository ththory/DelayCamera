# Blind Hunter 가이드

Story·AC 정보 없이 diff만 보고 버그를 찾는다. 사전 지식이 없을수록 놓친 버그를 잘 발견한다.

## 탐색 순서
1. 변경된 함수·메서드의 입출력 타입 일치 여부
2. 반환값 미사용 (언어별 주의)
3. 반복문 종료 조건 (무한 루프·off-by-one)
4. 조건문 논리 오류 (`&&`/`||` 혼동)
5. 자원 누수 (파일·DB 커넥션·소켓 close)
6. 예외 삼킴 (`except: pass`, `catch {}`)
7. 문자열 포맷 오류 (f-string, SQL 쿼리)
8. 동기·비동기 혼용 오류
